# Protocol and Port Reference

| Protocol | Port | Description                |
|----------|------|---------------------------|
| HTTP     | 80   | Web traffic (unencrypted) |
| HTTPS    | 443  | Secure web traffic        |
| SSH      | 22   | Secure shell              |
| Telnet   | 23   | Unencrypted remote access |
| FTP      | 21   | File Transfer Protocol    |
| SFTP     | 22   | Secure FTP (over SSH)     |
| SMTP     | 25   | Email sending             |
| POP3     | 110  | Email retrieval           |
| IMAP     | 143  | Email retrieval           |
| DNS      | 53   | Domain Name System        |
| DHCP     | 67/68| Dynamic Host Config       |
| SNMP     | 161  | Network management        |
| RDP      | 3389 | Remote Desktop Protocol   |
| NTP      | 123  | Network Time Protocol     |
| LDAP     | 389  | Directory services        |
| SMB      | 445  | Windows file sharing      |
| MySQL    | 3306 | Database                  |
| PostgreSQL|5432 | Database                  |

*Add more as needed for your scenarios!*
