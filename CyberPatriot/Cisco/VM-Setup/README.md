# Cisco Packet Tracer Scenario Setup

This directory contains unique Packet Tracer scenarios for CyberPatriot training.

## Scenario List

- **Basic_Network_Security.pkt**: Device hardening, password security, and SSH configuration.
- **Advanced_Network_Security.pkt**: VLANs, ACLs, and advanced service restrictions.
- **ACL_Challenge.pkt**: Focused on access control list creation and troubleshooting.

## Setup Instructions

1. Open the desired `.pkt` file in Cisco Packet Tracer.
2. Follow the scenario instructions in the "Scenario Notes" section of each file.
3. Complete all tasks and verify your configuration using `show` commands.
4. Save your work and document all changes for scoring.

*All scenarios are unique and designed for CyberPatriot.*
